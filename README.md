Project : Py-Ver.7

This project is a simple implementation of a chatbot using Python and the Natural Language Toolkit (NLTK) library. These chatbots can answer several predefined conversation patterns, such as greeting users, answering questions about their identity, and responding to breakups. Using pre-programmed rules, chatbots can interact with users through a simple text interface.
